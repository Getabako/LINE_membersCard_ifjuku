# member-card 要件定義書

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| アプリ名 | member-card |
| 業種 | オンライン学習塾 |
| プラットフォーム | LINEミニアプリ（LIFF） |

## 選択機能

### 1. デジタル会員証
- QRコード/バーコード表示
- 会員情報表示（氏名、会員番号、登録日）
- ポイント残高表示
- 会員ランク表示
- ポイント履歴

### 2. 予約管理
- カレンダーから日付選択
- 利用可能な時間枠表示
- 予約確認画面
- 予約変更・キャンセル機能
- 予約履歴
- リマインド通知（LINEメッセージ）

## 技術スタック

### フロントエンド
- React 18+
- TypeScript
- LIFF SDK
- Zustand（状態管理）
- Tailwind CSS（スタイリング）
- Vite（ビルドツール）

### バックエンド
- Node.js
- Express
- TypeScript
- Prisma（ORM）
- PostgreSQL（データベース）

### インフラ
- Vercel または Render（フロントエンド）
- Railway または Render（バックエンド）

## ユーザーストーリー

### 会員証機能
1. ユーザーとして、LINEから会員証を表示したい
2. ユーザーとして、自分のポイント残高を確認したい
3. ユーザーとして、会員ランクを確認したい
4. 店舗として、QRコードを読み取って会員を特定したい

### 予約機能
1. ユーザーとして、空いている日時を確認して予約したい
2. ユーザーとして、予約内容を確認・変更・キャンセルしたい
3. ユーザーとして、予約のリマインド通知を受け取りたい
4. 管理者として、予約状況を一覧で確認したい

## 画面一覧

1. **ホーム画面** - 会員証表示、メニュー
2. **会員証画面** - QRコード、会員情報詳細
3. **ポイント履歴画面** - ポイント取得・使用履歴
4. **予約画面** - カレンダー、時間枠選択
5. **予約確認画面** - 予約内容確認
6. **予約一覧画面** - 予約履歴

## 非機能要件

- レスポンシブ対応（モバイルファースト）
- ページ読み込み3秒以内
- LIFF認証によるセキュアなアクセス
- LINE Messaging APIによる通知
